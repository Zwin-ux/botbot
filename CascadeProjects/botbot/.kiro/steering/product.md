# Product Overview

BotBot is a Character-AI-style agent system that creates AI companions with persistent memory, operating across Discord and a web "Chao Garden" interface.

## Core Value Proposition

- **Natural Language First**: No slash commands - users talk naturally with AI agents
- **Persistent Memory**: Vector-based memory system that remembers facts, preferences, and events across sessions
- **Multi-Platform**: Seamless experience between Discord chat and web management interface
- **Customizable Personalities**: Agents have distinct personas, moods, and traits

## Key Features

- Discord bot with natural language processing for agent adoption and conversation
- Web garden interface for agent management and visualization
- Vector-based long-term memory with automatic extraction and decay
- Mood system using PAD model (Pleasure-Arousal-Dominance)
- Content moderation and rate limiting for safety
- Tool system foundation for extensible agent capabilities

## Target Users

- Discord communities wanting AI companions
- Developers interested in AI agent systems
- Users seeking personalized AI interactions with memory persistence

## Business Model

Open source project designed for self-hosting, with focus on community deployment and educational use.
